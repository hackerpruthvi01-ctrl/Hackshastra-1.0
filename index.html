Course Overview

What is Hackshastra 1.0?

Hackshastra 1.0 is a comprehensive 10-day program designed to introduce students to the world of ethical hacking and cybersecurity. This course provides hands-on experience with real-world tools and techniques used by security professionals.

Course Objectives

  • Understand cybersecurity fundamentals
  • Learn penetration testing methodologies
  • Master essential hacking tools (Termux, Nmap, Zphisher, Hydra)
  • Practice ethical hacking using Android phone
  • Prepare for cybersecurity certifications

10-Day Schedule

Complete hands-on learning with practical demonstrations

DAY 1

Introduction to Ethical Hacking

  • What is Ethical Hacking?
  • Types of Hackers (White/Grey/Black Hat)
  • Devices used for hacking
  • Tools needed (Android Phone, Termux, F-Droid, Hacker's Keyboard)
Theory Foundation
DAY 2

Setting Up Your Android Phone

  • Install Termux & F-Droid
  • Hacker's Keyboard setup
  • Update & upgrade packages
  • Install Linux packages (git, wget, curl, python, nano, etc.)
Bonus: Termux customisation using T-header tool
Termux F-Droid T-header
DAY 3

Network Scanning & Information Gathering

  • Install & use nmap in Termux
  • Network Discovery using ping, whois, dnsenum
  • Scan your WiFi and nearby devices
  • Network sniffing using http canary (No root)
Nmap Ping Whois DNSenum
DAY 4

Website Recon & Vulnerability Scanning

  • Install & use tools like sqlmap, WPSCAN (WordPress), Nikto
  • Discover vulnerabilities in test sites
SQLMap WPSCAN Nikto
DAY 5

Social Media Hacking Awareness + Phishing Intro

  • What is Social Engineering?
  • Real cases of social media account takeovers
  • Introduction to phishing: How it works
Social Engineering Phishing Theory
DAY 6

Practical Phishing using Android

  • Tool: Zphisher or HiddenEye in Termux
  • Facebook, Instagram, Gmail phishing pages
  • Tunneling with Ngrok or Serveo
  • Collecting credentials (demo only, on your own form)
Zphisher HiddenEye Ngrok Serveo
DAY 7

Password Cracking Basics

  • Wordlist attacks using Hydra
  • Bruteforce demo (on your own login pages)
  • Tools: Crunch, Hydra, Cupp
Hydra Crunch Cupp
DAY 8

Android Hacking Tools Overview

  • NetHunter Lite vs Full
  • Toolkits like AndroRAT (Remote Access)
  • Spy apps (for awareness & countermeasures)
NetHunter AndroRAT Android Tools
DAY 9

Protection Techniques & Anonymous Browsing

  • Use of VPN, Tor, ProxyChains on Android
  • Encrypting data
  • Detecting phishing, malware links
VPN Tor ProxyChains
DAY 10

Final Project + Certification

  • Task: Scan your WiFi + Test phishing page + Detect vulnerability on demo site
  • Submit findings in PDF
  • PDF certificate (custom-designed)
Project-Based Learning
All Tools Project Certificate
